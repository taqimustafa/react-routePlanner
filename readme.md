# Route Planner

## Quick Start
- Run Mongodb `mongod.exe`

- Go to terminal and in the project root execute:
  `node app.js` (this will start Express App)

- Go to terminal and in the project root execute:
  `webpack -w` (this will build React Application)

And You're done!
